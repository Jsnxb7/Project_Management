Phase 3 and Phase 4 Update
==========================

Added project-management feature layer:
- Milestones collection and milestone API routes.
- Project milestone creation and archive controls in Manage Project.
- Task-to-milestone relation with validation.
- Task deadline validation against both project and milestone deadlines.
- Checklist/subtask completion guard before marking a task Done.
- Comments, attachments, subtasks, and milestones now carry organization/project relation metadata where appropriate.
- Activity logs are added for milestone creation/update/archive and checklist/task events.

Added analytics and UI layer:
- Project Health card with risk score.
- Blocked task count, unassigned task count, milestone count.
- Milestone progress analytics.
- Workload/user completion progress bars.
- Warning Center includes overdue milestones.
- Task board UI includes milestone selectors for create/edit task forms.

Database setup changes:
- New milestones collection.
- New indexes for milestones and task milestone_id.

After deployment, run:
python mongo_setup.py
python scripts/repair_org_relations.py

Validation performed:
- Python compile check passed with python -m compileall.
- JavaScript syntax check passed with node --check for all static/js files.
